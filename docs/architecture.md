## Потоки

```text
Пользователь → titlo.ru (Next)
                 ├─ статические/SSR страницы

Вход / полный продукт → cabinet.titlo.ru
```

**Переходный деплой кабинета (май 2026):** код скопирован на новый VPS, MySQL остаётся на `178.250.157.140`. Подробно: [cabinet-servers.md](./cabinet-servers.md).

## Принципы

- **Laravel** — источник правды для данных, лимитов демо, регистрации.
- **Next** — маркетинг, SEO, интерактив демо, BFF-прокси (без секретов в браузере).
- **Битрикс / Kraken** — не используется в этом репозитории (миграция контента отдельно).

## Репозиторий

- `app/` — App Router, API routes
- `components/` — UI (по эталонам из `docs/examples/`)
- `lib/` — клиент lk, конфиг
- `docs/` — описание «что и как»
