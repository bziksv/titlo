## Часть 1. Универсальный каталог эффектов

Каждый приём: **что видно → зачем → сложность → куда на Датагон**.

Условные уровни сложности:

- **L** — CSS + немного React, без тяжёлого JS
- **M** — scroll/Intersection Observer, видео, слайдер
- **H** — кастомный курсор, WebGL, полноэкранные сцены

### 1. Hero и первый экран

| Эффект | Референсы | Зачем | Сложность | На Датагон |
|--------|-----------|-------|-----------|------------|
| Крупная типографика + подзаголовок-моно | Red Collar, AIC, ONY | Сразу «студийный» уровень, не «Битрикс-лендинг» | L | Уже есть в `*HERO`; усилить `text-[2.75rem]`, `font-mono` для `sub` |
| Showreel / видео в hero | Red Collar, AIC, CP (меню) | Эмоция, доверие | M | Короткий loop UI модуля (5–10 с) вместо статичного скрина |
| Scroll-driven сцена (глобус, 3D, «путешествие») | Red Collar [тамо](https://redcollar.ru/work/tamo/) | История продукта по скроллу | H | Упрощённо: **parallax** скринов LK в `HeroParallaxMedia` (уже есть компонент) |
| Ротатор аудитории / слов | AIC (`человека → бизнеса → государства`), Nota (`государства / бизнеса`) | Динамика без видео | M | Ротатор сценариев: «SEO · агентство · копирайтинг · e-commerce» |
| Кастомный курсор `drag` | Red Collar | Бренд, «мы не шаблон» | H | Только desktop; `cursor: grab` + лёгкий follower — опционально |
| Градиентный brand-hero | Bitrix24, текущие лендинги | Узнаваемость Датагон | L | Оставить `bg-brand-800` + `ParallaxMonoScene` |

## Часть 2. По сайтам (детально)

### Bitrix24 — [bitrix24.ru](https://www.bitrix24.ru/)

**Характер:** продуктовый маркетинг, плотная информация, «живой» UI в макетах.

**Эффекты для заимствования:**

1. **Переключатель блоков продукта** (Мессенджер / CRM / Задачи…) — табы с превью контента справа.
2. **Карточки возможностей** — иконка + заголовок + 1 строка, сетка 3–4 колонки.
3. **Встроенный mockup продукта** — чат AI с готовыми подсказками-кнопками.
4. **Счётчик масштаба** — крупная цифра + подпись.
5. **Вертикальные списки с галочками** внутри цветных секций.
6. **Мягкие градиенты** фона секций (не плоский белый).

**Не копировать:** перегруз меню, корпоративный «кислотный» маркетинг — у Датагон тон спокойнее.

## Часть 5. Технические ограничения Датагон

1. **Next.js Image** — скрины только из `/public/modules/assets/`, без hotlink.
2. **Производительность** — без тяжёлого WebGL на 18 страницах; `prefers-reduced-motion` отключать анимации.
3. **Единый бренд** — `brand-*`, не копировать чёрный Red Collar / фиолетовый Mish.
4. **Доступность** — marquee с `aria-hidden` или пауза по hover; табы с keyboard.
5. **Не ломать verify-скрипты** — текстовые якоря `MUST_HAVE` сохранять.
