# Кабинет — HTML-редактор (changelog)

Модуль: `/html-editor` · config `cabinet-html-editor.php` · badge **v1.5.4s**

## 1.5.4s — 2026-05-26

- **HTML-код:** подсветка синтаксиса (CodeMirror, тема neo) — номера строк, перенос, сворачивание тегов, автозакрытие, активная строка; синхронизация с CKEditor без изменений.
- **Проверка:** http://localhost:3002/edit-description/{id} — панель «HTML-код».

## 1.5.3s — 2026-05-25

- **Список проектов:** блок «Как пользоваться» — split-view, пресеты, публичная ссылка, поиск.
- **Публичная страница шаринга:** CTA регистрации с перечислением возможностей кабинета.
- **Проверка:** `/html-editor`, `/public/share/html-editor/{token}`.

## 1.5.2s — 2026-05-25

- **Экран редактирования:** убраны дубли (подсказка, блок «Проект / Начало текста», лишние «Назад»). Пресеты и публичная ссылка — в свёрнутых секциях. Проект в заголовке и хлебных крошках.
- **Проверка:** `/edit-description/{id}` — сразу виден редактор.

## 1.5.1s — 2026-05-25

- **Раскладка редактора:** переключатель «Рядом» / «Код снизу» — полная ширина, код под CKEditor; выбор сохраняется в localStorage.
- **Проверка:** `/edit-description/{id}` → «Код снизу».

## 1.5s — 2026-05-25

- **Публичная ссылка** на HTML-текст (как в анализе текста): `POST html-editor/public-share`, `GET /public/share/html-editor/{token}`, 30 дней, превью + HTML-код.
- **UI:** блок «Публичная ссылка без регистрации» на `/edit-description/{id}` — создать / обновить / отозвать / копировать.
- **БД:** `html_editor_public_shares`, cron очистки.
- **Проверка:** редактирование текста → «Создать публичную ссылку» → URL в инкогнито **200**.

## 1.4.2s — 2026-05-25

- **Навигация:** «← Мои проекты» в шапке карточки и сверху формы + хлебные крошки (html-редактор → проект → экран).
- **Возврат к проекту:** с редактирования текста/проекта — `/html-editor?project={id}` открывает нужный проект в списке.
- **Проверка:** `/edit-description/{id}` → «Мои проекты».

- **Проверка:** форма текста → «Сохранить как пресет» → «11111» → кнопка в «Мои пресеты».

## 1.4s — 2026-05-25

- **Формы:** create/edit project, create/edit text — единый `cabinet-html-editor.css`, badge в шапке.
- **Редактор:** CKEditor (как раньше), общий partial `cabinet-html-editor-ckeditor`.
